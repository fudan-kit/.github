# 🎓 Fudan Templates

A curated collection of high-quality starter templates and boilerplates for students and developers at Fudan University.

## 🚀 What’s inside

* ⚛️ Frontend templates (React, Next.js, etc.)
* 🐍 Python & Data Science
* 🤖 AI / Machine Learning
* 🧱 Backend services
* 📚 Course project starters

## 🎯 Goal

Make it easier to start projects, learn faster, and build better.

## 🤝 Contributing

Contributions are welcome! Feel free to open issues or submit pull requests.

## 📌 Naming Convention

All repositories follow:

```
<tech>-template
```

Examples:

* react-template
* pytorch-template
* backend-template
